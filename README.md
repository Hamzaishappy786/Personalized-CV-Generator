# Cheentapakdumdum Resume

A resume generator built for developers. Pick your track, select your skills, fill in your experience, and get a print-ready resume in minutes.

**Live at:** https://personalized-cv-generator.vercel.app/

---

## How to use

Just open the link above in your browser. No account, no install, no setup.

---

## Features

- 16 developer tracks (Web, Mobile, DevOps, ML/AI, Blockchain, and more)
- 3 resume templates: Modern Classic, Developer Mono, and Sidebar
- AI-powered bullet point enhancement via Hugging Face (optional)
- Dark mode
- Print to PDF with A4 layout

---

## Run locally

If you want to run it on your own machine:

```
git clone https://github.com/Hamzaishappy786/Personalized-CV-Generator.git
cd Personalized-CV-Generator
npm install
npm run dev
```

Then open http://localhost:5173 in your browser.

---

## Tech stack

- React 18 + TypeScript + Vite
- Tailwind CSS v4
- Zustand
- React Router v6
- Hugging Face Inference API (optional, for AI features)
